
# WebView APP built with 💙 with Flutter 🎯

### Get Started

```shell
git clone https://github.com/Syed-Muhammad-Ali/WebView_APP.git
```

```shell
flutter clean
```

```shell
flutter packages get
```

```shell
flutter run
```
